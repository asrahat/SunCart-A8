# 🛒 SunCart-A8

## 📌 Project Overview
SunCart-A8 is a modern and responsive e-commerce web application built to deliver a smooth online shopping experience. It focuses on clean UI design, fast performance, and dynamic product rendering using modern frontend technologies.

---

🔗 https://sun-cart-a8-ez2o.vercel.app/

---
## 📁 Repository
🔗 https://github.com/asrahat/SunCart-A8
---

## 🎯 Purpose
The goal of this project is to build a scalable and user-friendly shopping platform where users can:
- Browse products easily
- View detailed product information
- Experience fast and responsive navigation
- Enjoy a modern UI/UX design

---

## ✨ Key Features
- 🛍️ Dynamic product listing
- 🔍 Product details page
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚡ Fast loading with optimized performance
- 🧭 Clean and modern navigation
- 🔄 Smooth routing with Next.js
